**Описание:**
Простое приложение для конвертации валют с графическим интерфейсом. Использует внешний API для получения актуальных курсов. Сохраняет историю операций в JSON-файл.

**Как получить API-ключ:**
1. Зарегистрируйтесь на сайте [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Получите бесплатный или платный API-ключ.
3. Вставьте ключ в переменную `API_KEY` в файле `currency_converter.py`.

**Как использовать:**
1. Установите Python и pip.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python currency_converter.py`
4. Выберите валюты «Из» и «В», введите сумму и нажмите «Конвертировать».
5. Результат появится под кнопкой.
6. История операций отображается в таблице.
   Возможные тесты (примеры использования):
Конвертировать 100 USD в EUR — результат должен быть корректным по текущему курсу.
Ввести отрицательную сумму — появится ошибка.
Попробовать конвертировать без интернет-соединения — появится ошибка API.
Закрыть и открыть приложение — история должна сохраниться и загрузиться.
